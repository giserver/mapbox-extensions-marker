## usage  

### install
#### npm
```shell
npm install mapbox-extensions-marker
```

#### yarn
```shell
yarn add mapbox-extensions-marker
```

### code

``` ts
import { MarkerControl } from 'mapbox-extensions-marker'
import 'mapbox-extensions-marker/dist/index.css'

map.addControl(new MarkerControl({}));
```
